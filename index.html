FAQ

How do I install it?

Download the Windows installer and run it. Since it is from a unknown publisher, you may need to allow it to run. You can click on More info and then Run anyway. Then follow the on-screen steps to install.

Does it work on Linux?

Officially only Windows is supported; Linux may require compatibility tools like Wine.

Is my Gemini API key safe?

Yes, it is safe. It is stored in the app's configuration file.

Why shouldn't I save it in C:/Program Files?

Program files is a system folder which is highly protected by Windows. So writeable files are not allowed to be saved here. It is recommended to save it in C:/Users/your_username or D: or E:
